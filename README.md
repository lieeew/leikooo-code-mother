# Code Mother

AI-powered code generation platform.

## Features

- Multiple code generation modes (HTML, Multi-file, Vue project)
- Tool call tracking and persistence
- Chat history management
- Vue project scaffolding

## Tech Stack

- Spring Boot 3.x
- Spring AI
- Vue 3
- MySQL + MyBatis-Plus
- Redis

## Getting Started

```bash
# Backend
cd code-mother
mvn spring-boot:run

# Frontend
cd code-mother-fronted
npm install
npm run dev
```
